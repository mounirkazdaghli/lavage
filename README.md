# L'Expert Eco Lav - Car Wash Reservation System

A modern web application for booking and managing car wash services with dedicated admin and user interfaces.

## Features

### 🌟 Admin Dashboard
- **Reservations Tab**: View all customer reservations
- **Planning Tab**: Manage staff schedules and service capacity
- **My Reservations Tab**: Track admin's own operations

### 👤 User Dashboard
- **Reservations Tab**: Book and manage personal reservations

### 🛠️ Services
- Eco-friendly car washing
- Premium detailing
- Protection coatings
- Interior cleaning

## Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express
- **Database**: MongoDB
- **Authentication**: JWT
- **Deployment**: Vercel/Netlify
